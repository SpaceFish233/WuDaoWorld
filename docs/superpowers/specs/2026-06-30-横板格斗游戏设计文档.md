# 横板格斗游戏设计文档

## 项目概述

### 项目名称
武道天下（暂定）

### 项目类型
横板格斗2D游戏

### 技术栈
- 游戏引擎：Phaser 3
- 开发语言：TypeScript
- 构建工具：Vite
- 版本控制：Git

### 项目目标
开发一款像素风格的横板格斗游戏，支持单人闯关、双人闯关和双人对战三种模式，包含完整的装备系统和角色成长系统。

#### 1.3 核心概念

**组件化架构（ECS 变体）：**
- **实体（Entity）**：玩家、敌人等游戏对象，只是组件的容器
- **组件（Component）**：纯数据，如生命值、攻击力、位置等
- **系统（System）**：处理逻辑，如战斗系统处理所有攻击组件

**优点：**
1. 高度解耦：修改装备系统不影响战斗系统
2. 易于扩展：添加新角色只需添加配置数据
3. 复用性强：组件可在不同实体间共享

#### 2.3 伤害计算公式

```typescript
// 基础伤害计算
基础伤害 = 攻击者攻击力 × 技能倍率

// 防御减伤
防御减伤 = 防御者防御力 / (防御者防御力 + 100)

// 最终伤害
最终伤害 = 基础伤害 × (1 - 防御减伤) × 暴击倍率 × 装备加成

// 暴击判定
暴击率 = 基础暴击率 + 装备暴击率
暴击倍率 = 1.5 + 装备暴击伤害加成
```

```typescript
class SaveManager {
  private static readonly SAVE_KEY = 'wudao_tianxia_save';
  private static readonly MAX_SAVES = 3;  // 最大存档数
  
  // 保存游戏
  static save(saveData: GameSave, slot: number): void {
    const key = `${this.SAVE_KEY}_${slot}`;
    localStorage.setItem(key, JSON.stringify(saveData));
  }
  
  // 读取游戏
  static load(slot: number): GameSave | null {
    const key = `${this.SAVE_KEY}_${slot}`;
    const data = localStorage.getItem(key);
    return data ? JSON.parse(data) : null;
  }
  
  // 获取所有存档
  static getAllSaves(): (GameSave | null)[] {
    const saves: (GameSave | null)[] = [];
    for (let i = 0; i < this.MAX_SAVES; i++) {
      saves.push(this.load(i));
    }
    return saves;
  }
  
  // 删除存档
  static deleteSave(slot: number): void {
    const key = `${this.SAVE_KEY}_${slot}`;
    localStorage.removeItem(key);
  }
}
```

```typescript
class AudioManager {
  private static instance: AudioManager;
  private bgmVolume: number = 0.7;
  private sfxVolume: number = 1.0;
  private currentBGM: Phaser.Sound.BaseSound | null = null;
  
  static getInstance(): AudioManager {
    if (!AudioManager.instance) {
      AudioManager.instance = new AudioManager();
    }
    return AudioManager.instance;
  }
  
  // 播放背景音乐
  playBGM(scene: Phaser.Scene, key: string, loop: boolean = true): void {
    this.stopBGM();
    this.currentBGM = scene.sound.add(key, {
      volume: this.bgmVolume,
      loop: loop,
    });
    this.currentBGM.play();
  }
  
  // 停止背景音乐
  stopBGM(): void {
    if (this.currentBGM) {
      this.currentBGM.stop();
      this.currentBGM = null;
    }
  }
  
  // 播放音效
  playSFX(scene: Phaser.Scene, key: string): void {
    scene.sound.play(key, {
      volume: this.sfxVolume,
    });
  }
  
  // 设置音量
  setBGMVolume(volume: number): void {
    this.bgmVolume = Math.max(0, Math.min(1, volume));
    if (this.currentBGM) {
      (this.currentBGM as any).setVolume(this.bgmVolume);
    }
  }
  
  setSFXVolume(volume: number): void {
    this.sfxVolume = Math.max(0, Math.min(1, volume));
  }
}
```

```typescript
// src/core/EventBus.ts
export class EventBus {
  private static instance: EventBus;
  private listeners: Map<string, Function[]> = new Map();

  static getInstance(): EventBus {
    if (!EventBus.instance) {
      EventBus.instance = new EventBus();
    }
    return EventBus.instance;
  }

  on(event: string, callback: Function): void {
    if (!this.listeners.has(event)) {
      this.listeners.set(event, []);
    }
    this.listeners.get(event)!.push(callback);
  }

  off(event: string, callback: Function): void {
    const callbacks = this.listeners.get(event);
    if (callbacks) {
      const index = callbacks.indexOf(callback);
      if (index > -1) {
        callbacks.splice(index, 1);
      }
    }
  }

  emit(event: string, ...args: any[]): void {
    const callbacks = this.listeners.get(event);
    if (callbacks) {
      callbacks.forEach(callback => callback(...args));
    }
  }
}
```

```typescript
// src/systems/CombatSystem.ts
import { EventBus } from '../core/EventBus';
import { AttackComponent } from '../components/AttackComponent';
import { DefenseComponent } from '../components/DefenseComponent';
import { HealthComponent } from '../components/HealthComponent';

export class CombatSystem {
  private eventBus: EventBus;

  constructor() {
    this.eventBus = EventBus.getInstance();
    this.setupEventListeners();
  }

  private setupEventListeners(): void {
    this.eventBus.on('attack', this.handleAttack.bind(this));
  }

  private handleAttack(
    attacker: { attack: AttackComponent; position: { x: number; y: number } },
    defender: { defense: DefenseComponent; health: HealthComponent; position: { x: number; y: number } }
  ): void {
    // 检查攻击范围
    const distance = Math.abs(attacker.position.x - defender.position.x);
    if (distance > attacker.attack.attackRange) {
      return;
    }

    // 计算伤害
    let damage = attacker.attack.baseDamage;

    // 暴击判定
    const isCritical = Math.random() < attacker.attack.criticalRate;
    if (isCritical) {
      damage *= 1.5 + attacker.attack.criticalDamage;
    }

    // 防御减伤
    const defenseReduction = defender.defense.defense / (defender.defense.defense + 100);
    damage *= (1 - defenseReduction);

    // 格挡判定
    if (defender.defense.isBlocking) {
      damage *= 0.3; // 格挡减少70%伤害
    }

    // 应用伤害
    defender.health.currentHP -= damage;

    // 触发事件
    this.eventBus.emit('damageDealt', {
      attacker: attacker,
      defender: defender,
      damage: damage,
      isCritical: isCritical,
    });

    // 检查死亡
    if (defender.health.currentHP <= 0) {
      defender.health.isDead = true;
      this.eventBus.emit('entityDeath', defender);
    }
  }
}
```

## 风险评估

### 技术风险

1. **Phaser 3 学习曲线**
   - 风险：不熟悉 Phaser 3 可能导致开发进度缓慢
   - 缓解：提前学习 Phaser 3 文档和示例

2. **本地双人输入处理**
   - 风险：键盘冲突可能导致输入问题
   - 缓解：合理设计键位映射，使用 Phaser 输入系统

3. **性能问题**
   - 风险：大量实体和特效可能导致性能问题
   - 缓解：使用对象池、优化渲染、限制实体数量

### 内容风险

1. **素材资源获取**
   - 风险：可能找不到完全符合需求的素材
   - 缓解：使用多种开源资源，必要时进行修改

2. **游戏平衡性**
   - 风险：角色和装备可能不平衡
   - 缓解：进行充分测试，调整数值配置
